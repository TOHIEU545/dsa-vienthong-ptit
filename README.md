# PTIT DSA

Repository lưu các bài tập ôn luyện **DSA** theo từng chủ đề.

## Phân loại bài tập

| Mã | Chủ đề |
|---|---|
| `CON1_x` | Sinh |
| `CON2_x` | Quay lui |
| `CON3_x` | Tham lam |
| `CON4_x` | Chia để trị |
| `CON5_x` | Quy hoạch động |
| `CON6_x` | Sắp xếp |
| `CON7_x` | Stack |
| `CON8_x` | Queue |
| `CON9_x` | Đồ thị |

Trong đó `x` là số thứ tự của bài.
